# 日本の四季を英語で表現する  
*How to Describe Japan’s Four Seasons in English*

## 概要
> Japan has four distinct seasons, each celebrated with seasonal foods, festivals, and scenery.

## 表現の例
- **Spring**: *cherry blossoms*, *fresh greenery*  
- **Summer**: *fireworks*, *summer festivals*  
- **Autumn**: *autumn leaves*, *harvest*  
- **Winter**: *snow festivals*, *New Year*
