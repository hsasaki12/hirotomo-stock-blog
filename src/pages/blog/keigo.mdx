# 日本語の「敬語」を英語で説明する  
*How to Explain Keigo in English*

## Keigoとは
日本語の敬語は **尊敬語 / 謙譲語 / 丁寧語** の3種類。目的は **敬意の表現と距離感の調整** です。

> Keigo is a system of honorific speech in Japanese used to show respect and manage social distance.

## 種類と例
- **尊敬語（respectful language）**：相手を高める  
  - 例：行く → *いらっしゃる* / *おいでになる*  
- **謙譲語（humble language）**：自分を低める  
  - 例：行く → *伺う* / *参る*  
- **丁寧語（polite language）**：語尾の丁寧化  
  - 例：〜です / 〜ます

## 英語での説明ポイント
- 「身分差」ではなく**場面**での切り替え  
- **社外/社内**の使い分け（*customer-facing tone* に近い）  
- 直訳より **意味を補って説明** する
